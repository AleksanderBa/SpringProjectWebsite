================================================================================
  NATURAL HISTORY MUSEUM AARHUS - WEBSITE PROJECT
  Get Started with Visual Studio Code
================================================================================

Hello! This project contains a complete website for the Natural History Museum
Aarhus (Naturhistorisk Museum). Follow these simple steps to get it running.

================================================================================
  STEP 1: OPEN IN VISUAL STUDIO CODE
================================================================================

1. Open Visual Studio Code
2. Click: File > Open Folder
3. Select this project folder
4. Click "Select Folder"

================================================================================
  STEP 2: INSTALL DEPENDENCIES
================================================================================

1. Open Terminal in VS Code:
   - Menu: Terminal > New Terminal
   - Or press: Ctrl + ` (backtick key)

2. Type this command and press Enter:
   
   npm install

3. Wait for installation to complete (may take 2-5 minutes)

================================================================================
  STEP 3: SET UP IMAGE FOLDERS
================================================================================

1. In VS Code, right-click in the file explorer (left sidebar)
2. Click "New Folder"
3. Name it: public
4. Right-click the "public" folder
5. Click "New Folder"
6. Name it: assets

Your folder structure should now look like:

================================================================================
  STEP 4: START THE WEBSITE
================================================================================

   npm run dev

================================================================================
  TROUBLESHOOTING
================================================================================

Problem: "Command not found: npm"
Solution: Install Node.js from https://nodejs.org/

Problem: "Port already in use"
Solution: Press Ctrl + C to stop server, then run "npm run dev" again

Problem: Images not showing
Solution: Make sure images are in public/assets/ folder

Problem: Changes not appearing
Solution: Save the file (Ctrl + S) and hard refresh browser (Ctrl + Shift + R)
